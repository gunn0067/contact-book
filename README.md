# 📒Contact Book (CLI)

A command-line contact management application built in Python that allows users to add, view, search and delete contacts.

---

## 📌Description

This project is a fully functional contact book that runs in the terminal. It stores contacts as a list of dictionaries, each containing a name, phone number and email address. Users can perform all basic CRUD operations through a simple menu-driven interface. Built as part of a Python internship project to practice data structures, functions and CLI application design.

---

## ✨Features

- Add new contacts with name, phone and email
- View all saved contacts in a numbered list
- Search contacts by name (partial match supported)
- Delete contacts by name
- Duplicate contact detection
- Input validation and clean error messages
